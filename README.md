# Ucenje — Number 19 in the Quran

Personal scholarly research project by Alem Basic. Empirical study of mathematical structure attributed to the number 19 in the Quran, plus 19-TET (equal temperament) sonification of Quranic counts.

**Live:** https://ucenje.alai.no

## Audience
Multi-level — designed for readers from age 10 (`mladi.html`) to mathematical physicists (`matematika.html`), Muslims and non-Muslims alike.

## Structure
- `index.html` — landing
- `bs.html` / `en.html` / `no.html` — full multilingual deep-dive
- `matematika.html` — for mathematicians
- `teologija.html` — for theologians
- `narod.html` — general audience
- `mladi.html` — youth-oriented
- `laylatul-qadr.html` — astronomical sign (perigee + Metonic cycle argument)
- `pcele.html` — An-Nahl reference
- `19-explorer/` — interactive 6×19 visualization
- `audio/` — 19-TET sonification + poetic intro
- `assets/` — fonts (EB Garamond, Amiri, IBM Plex Mono), CSS, animations

See `DESIGN-SPEC.md` for typography, palette, animation vocabulary.

## Status
Work in progress. Not affiliated with any commercial entity. Reading-first; not a product.
